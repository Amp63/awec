# Another WorldEdit Clone

A generic WorldEdit clone for BTA. Works as a client or server mod.

## Current Feature List
- `//set`
- `//copy` and `//paste`
- `//replace`
- `//up`
- Block patterns (like `50%stone,50%dirt`)
- Configurable permissions via `/wewhitelist`
- Toggleable edit wand via `/togglewand`


## Planned Features
- Shapes
  - Sphere, Cylinder, Pyramid, and their hollow variants
- Schematics
- Masks
- Navigation commands
  - thru, ascend, descend, jumpto, etc.
